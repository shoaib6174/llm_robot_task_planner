## Decisions Made
| Decision | Rationale |
|----------|-----------|
| Use niro-1 instead of niro-2 | User preference; RTX 5090 + 125GB RAM |
| ROS 2 Jazzy (not Humble) | Ubuntu 24.04 doesn't support Humble; Jazzy is Tier 1 for Noble |
| Gazebo Harmonic (not Fortress) | Harmonic is the Jazzy-paired Gazebo version |
| Native install (not Docker/OS downgrade) | GUI-heavy project; Docker adds GPU+X11 complexity; downgrade risks GPU drivers |
| venv with --system-site-packages | Required for rclpy access from venv; pip packages still isolated |
| PyTorch cu128 (not cu124/cu126) | RTX 5090 is Blackwell (sm_120); only cu128 has native support without warnings |
| Skip pip pyyaml | ROS 2 provides python3-yaml via apt, avoids version conflict |
