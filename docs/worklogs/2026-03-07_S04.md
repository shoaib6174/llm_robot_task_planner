### URDF Modifications
- Fixed mesh paths in `car_mecanum.urdf.xacro`: changed `file://$(find jetrover_description)` to `package://jetrover_description` for Gazebo mesh resolution

### Launch File

## Decisions Made
| Decision | Rationale |
|----------|-----------|
| D011: Embed jetrover_description inside our package (not separate) | Avoids managing two packages; COLCON_IGNORE + data_files + GZ_SIM_RESOURCE_PATH solves mesh resolution |
| D012: Use $(find pkg) not $(dirname) in xacro includes | $(dirname) fails when xacro is processed in launch context (no file info available) |
| D013: DiffDrive instead of MecanumDrive plugin | MecanumDrive requires angled friction setup; DiffDrive simpler for initial testing per PRD risk mitigation |
| D014: 3cm cubes (down from 6cm) | Gripper finger separation ~28mm; 3cm cubes are ~75% of max opening, ideal for reliable grasping |
| D015: Helper launch.sh script on niro-1 | colcon `source install/setup.bash` doesn't work via SSH bash -c; explicit AMENT_PREFIX_PATH export as workaround |
